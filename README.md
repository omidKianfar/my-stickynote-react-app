# Usage technologies

1. HTML 5
2. Sass Module
3. Flexbox
4. React
   1. ES6
   2. Jsx
   3. Condition
   4. Event handler
   5. Function Component
   6. Fragment
   7. Hooks:
      useState,
      useRef,
      useEffect,
      useContext
5. React-router
6. React-draggable

# Install

  1. Clone project

    https://github.com/omidKianfar/my-stickynote-react-app.git

  2. Install dependencies

    npm install

  3. Run project

    npm start

  4. Run json server

    npx json-server -w db.json -p=3050
